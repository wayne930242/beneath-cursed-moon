# 扮演書

當團隊大致決定想玩什麼風格後，每位玩家各自選擇一份扮演書。扮演書會告訴你：你扮演的是哪種角色，以及該角色擅長什麼。

雖然多人可以選同一份扮演書，但並不建議。不同扮演書能讓每位成員有清楚專長，也比較能輪流發揮。

本作扮演書如下：

- **勇士（Champion）**：以堅不可摧的護甲與意志守護無辜者的虔誠衛士。
- **調換兒（Changeling）**：受妖精廷院影響、身帶異界魅惑的流浪者。
- **幻夢者（Dreamer）**：心智受古異洞見侵染、精通夢與惡夢之星界巫術的施法者。
- **惡魔（Fiend）**：試圖救贖自我的惡魔，必須與自身邪性抗衡，仍能操使地獄之力。
- **獵人（Hunter）**：老練的怪物獵手，武藝高強且總能備齊所需工具。
- **死靈法師（Necromancer）**：掌控死亡魔法，可抽取生命並驅使不死者。
- **學者（Scholar）**：準備周全的智者，擅長辨識怪物弱點並攜帶各式實用道具。
- **惡棍（Scoundrel）**：前盜匪或亡命者，以狡智、髒招與魅力對抗黑暗。
- **邪術士（Warlock）**：精通地獄魔法，以靈魂換取焚滅敵人的力量。
- **吸血鬼（Vampire）**：受不死詛咒的夜之掠食者，必須以生者之血維生。
- **狼人（Werewolf）**：受狼化詛咒者，時刻與體內兇獸爭奪主導權。

每份扮演書都由相似結構組成。

## 動作與加值

扮演書會列出你各項動作的加值。當你使用該動作擲骰時，就把扮演書對應加值加上去。某些動作可能共用相同加值，例如理解戰況與通曉傳說。

## 能力

扮演書能力分為：

- **基礎能力（Basic）**：開局全部獲得。
- **進階能力（Advanced）**：開局可選兩項。

MC 也可以依劇本需求，讓角色以更資深狀態開局（例如起始多拿幾項進階能力）。隨冒險推進，角色通常每經歷數次冒險可再獲得新的進階能力。

## 裝備

有些扮演書會列出裝備，有些不會。這不代表沒列出的角色是空手上場。所有角色都預設擁有基本日常物資（衣物、食物、火把、繩索、金錢等）。

若武器沒有被條目明確列出，通常最多只能造成 2 傷害。防具也是同理：若不是扮演書或規則明確提供的護甲，對 PC 將面對的核心威脅通常不提供機制上的減傷優勢。

## 快速背景

每份扮演書也附有「快速背景」選項。若你想快速建立角色，或在 one-shot 臨時成角，這些內容很有幫助。

快速背景只是建議，不是限制。你可以自由調整，讓角色更貼近你想玩的樣子。
