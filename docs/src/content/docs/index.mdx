## 系統簡介

**Beneath Cursed Moon（詛咒月影）** 是一套聚焦於詭異氛圍與生存抉擇的桌上角色扮演遊戲。
玩家將在被詛咒籠罩的世界中，透過調查、資源管理、戰術互動與風險承擔，逐步揭露真相並活下來。
