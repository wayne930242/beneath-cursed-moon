# 怪物

本作將怪物分為五大類。分類本身通常不直接提供機制效果（除非特定扮演書如獵人會引用），主要目的在於把同主題怪物聚在一起，方便同一冒險內做組合。

同類怪物常共享部分弱點與防護需求，但不是絕對。若怪物對某武器材質／屬性會降護甲（火、鐵、銀、靈魂繫結），通常對應的防護工具也能驅離它。若怪物沒有這些弱點，防護工具通常無效（但部分扮演書能力可例外處理）。

本章也納入「人物（People）」條目，雖不屬怪物，卻是冒險中常見對象（待救平民、地方民兵、黑暗邪教成員等），且通常不具怪物型魔法弱點。

## 五大怪物類型

### 畸變體（Abominations）

來自異界，或受異界感染變異而成。型態多樣且扭曲腐敗。它們的弱點通常源於其關聯屬性：與水與月高度相關者常怕銀；許多目標也需要火焰焚燒才能真正殺死。

### 野獸（Beasts）

所有獸性生物：真實動物（狼、熊）與奇幻獸（如獅鷲）都可歸類。最關鍵的是受詛咒狼化者。多數野獸沒有明確可驅離弱點，狼人系是少數對銀高度脆弱者。

### 惡魔（Demons）

地獄孕生，殘酷且喜歡腐化善意。惡魔不只樂於傷害凡人，更樂於挑撥凡人互相傷害。它們會被靈魂繫結武器有效打擊，也因此會爭奪靈魂來對抗同類。

### 妖精（Fey）

居於深荒之地的妖精廷院。它們不朽、疏離，常把凡人當可拋棄玩具。妖精受廷院規律束縛，且可談判，但其契約充滿文字陷阱。鐵通常能有效克制與驅離妖精。

### 不死族（Undead）

死亡後仍「留在世界上」的存在。從無智骷髏、殭屍，到狡詐吸血鬼都屬此類。保有理性的個體多受強烈執念驅動（進食、復仇等）。不死者通常是「被製造」而非自然誕生：可能來自咬轉、死靈術、或區域怨念匯聚。整體上對靈魂繫結武器較脆弱。

### 其他怪物（Other Monsters）

無法歸入前述分類者。數量較少、共通性低。常見為魔法驅動造物，或任何你世界裡無法歸類但有戲劇價值的異常存在。

## 威脅等級

除類型外，怪物還分三種戰力層級：

- **僕役（Minions）**
- **衛士（Warriors）**
- **領主（Lords）**

這個等級用於估算：

- 一場戰鬥可投入數量
- 擊倒所需傷害
- 護甲值
- 攻擊傷害
- 攻擊效果數量
- 抵抗黑魔法對其生效難度

### 僕役（Minions）

邪惡軍勢中的基層單位。單體威脅不高，甚至可能被有武器的平民勉強對付；但成群時危險度大幅上升。

常見參考值：

- 擊倒需求：`2-4 傷害`
- 護甲：`0`（或 1，但可被對應材質繞過）
- 攻擊：`1-2 傷害`
- 攻擊效果：`1`

群聚規則：

- 3 名僕役協同可額外 `+1 攻擊效果`
- 5 名僕役協同可額外 `+2 攻擊效果`

### 衛士（Warriors）

單體即可構成實質威脅，強度大致接近一名 PC。單一衛士在多人 PC 圍攻下通常會被壓制，但對落單或分散 PC 極具殺傷。

常見參考值：

- 擊倒需求：`10+ 傷害`
- 護甲：`1`（或 2，且可被對應材質繞過）
- 攻擊：`3-4 傷害`
- 攻擊效果：`2`

### 領主（Lords）

怪物中的最高威脅。即使單獨一隻，也足以挑戰資深獵手團隊，且常伴隨僕役出場。

常見參考值：

- 擊倒需求：`15+ 傷害`
- 護甲：至少 `2`（或 3，遇對應材質時大幅下降）
- 攻擊：至少 `4 傷害`
- 攻擊效果：通常 `2`

面對領主前，玩家通常必須先完成充分調查與準備。

## 怪物條目如何使用

怪物數值通常只列出「戰鬥與其直接後果」最關鍵資訊。其餘細節（飛行速度、變身條件、某吸血鬼會哪些儀式法術等）交由遊戲主持人 MC 依劇情需求決定。

## 攻擊效果（Attack Effects）

怪物攻擊時可同時選用多個攻擊效果，類似 PC 在浴血奮戰高骰時可拿多選項。可用數量取決於怪物強度；極端情況下（PC 已陷死局、怪物極端暴怒等）可酌量加一個效果。

實務上，除非怪物真的無法穿透 PC 護甲，或戰局明顯崩盤，否則不建議只堆最大傷害。推位、擄人、拆場景、奪物等行為通常更有戲，也更能產生後續後果。

常見攻擊效果如下：

- 依攻擊條目造成傷害
- 額外 `+1 傷害`
- 攻擊獲得 `group` 標籤（同時打多目標）
- 令目標失衡（`-1 單次加值`）
- 到下次行動前所受傷害 `-1`
- 抓取／纏縛／擄走目標
- 奪取關鍵物件
- 破壞關鍵物件
- 強制移動目標到怪物想要的位置
