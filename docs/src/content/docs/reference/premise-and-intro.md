# 前提

《詛咒月影》是一款桌上角色扮演遊戲。你們將在黑暗奇幻世界中追獵怪物、斬除威脅，並保護那些可能成為獵物的人。它深受哥德奇幻作品影響，例如《惡魔城》（Castlevania）、《血源詛咒》（Bloodborne）與《地獄怪客》（Hellboy）。

雖然遊戲採用了許多恐怖元素，但它不是純恐怖遊戲，也不是「黑暗絕望到底」的 grimdark。怪物確實存在，但你的英雄也有能力擊敗它們；你們的任務是守住人們。你不該害怕黑暗，而是讓黑暗害怕你。

遊戲由一位 MC（Master of Ceremonies，主持人）與任意數量的 PC（玩家角色）進行。MC 負責扮演世界中的怪物與其他角色，PC 則各自扮演自己的英雄。雙方透過角色行動共同推進故事，重點不是彼此對抗，而是一起創造有張力、有挑戰的冒險。

典型流程通常是：得知威脅、查出怪物身分、做好作戰準備，最後保護受害者並狩獵怪物。有時你會一路追殺德古拉的爪牙，直抵華麗而詭譎的魔法城堡；有時你則得在村莊中找出誰其實是偽裝成人的狼人。調查與戰鬥會交錯發生，解謎能力與獵殺能力同樣重要。

本作沒有硬性規定唯一世界觀，但預設世界是黑暗奇幻風格：小村莊彼此被危險荒野隔開，大城市則成為社會型掠食者隱匿的溫床。細節由你們決定，書中會提供足夠的靈感素材。

## 如果你熟悉 Powered by the Apocalypse

如果你玩過 PbtA 遊戲，這裡會有很多你熟悉的元素，也有一些關鍵差異。像是 `2d6+能力`、`7-9` 的混合成功，依然是核心手感。

差別在於：《詛咒月影》假設壞事本來就正在發生。當你開始擲骰時，壓力已經存在，不管是當下威脅還是長期惡化。混合成功不一定每次都要額外塞進「新的懲罰」；很多時候，情境本身就已經是代價，例如「你沒拿到足夠資訊，現在要立刻行動，還是再拖時間讓更多人受害？」

這個系統鼓勵持續推進，而不是把失敗變成停滯。玩家不該因為主動行動而感到被懲罰；故事始終向前，就算「前進」不是由角色直接主導，也依然成立。

## 作者筆記

謝謝你翻開這本書。光是讀到這裡，我都非常感激。這本作品源自我對《惡魔城》（Castlevania）的熱愛，也源自我想在 TRPG 裡實現類似風格的願望。

《詛咒月影》在 itch.io 採自由付費。把它做成免費作品對我很重要，因為我希望更多人能在這些想法上繼續創作。任何支持我都很感謝，那些支持會用在更多美術、編輯與協作者回饋上。

我也想明確說明：這款遊戲不追求複製現實中各種偏見。種族歧視、性別歧視、恐同、恐跨與其他仇恨內容，不是這款遊戲要提供的樂趣。我們在這裡是為了享受「追獵怪物」這件事。

當然，恐怖題材難免觸及敏感意象，因此實際遊玩時，請依照你和玩家的舒適程度調整內容強度。你們完全可以把某些主題淡化、抽象化，或直接排除。

本作主要取材自歐洲神話與相關文學。這一方面是我熟悉的領域，一方面也是製作規模與研究時間的現實考量。若你後續寫出更適合其他地域、時代的改版內容，歡迎分享，我很樂意幫忙宣傳。
