# 其他動作

本區動作通常不限定於戰鬥或調查，可跨情境使用。

## 說服（Convince）

當你與平民交涉，想讓對方幫忙或取得某物時，擲說服。

- `6 以下`：對方拒絕協助。
- `7-9`：對方願意幫，但會要求交換、在不利情境提到你，或一遇危險就撤。
- `10-12`：你得到想要的配合或資源。
- `13+`：對方真正認同你的目標，甚至會主動擴大協助。

一般來說，同一情境下，平民通常只會被同一位 PC 說服一次。

## 大膽行動（Daring Action）

當你進行危險且需要身體力行的動作時，擲此動作，例如飛越屋頂、穿林逃命、衝入火場救人等。

- `6 以下`：目標失敗，且發生壞事。
- `7-9`：目標達成，但同時有代價。
- `10-12`：順利完成，無額外代價。
- `13+`：除成功外，還獲得額外好處。

常見代價（由 MC 依情境裁定）：

- 受到 1-2 傷害（無視護甲）
- 掉落、失去、損壞或耗盡重要物品
- 落入不利位置，恢復前承受 `-1 單次加值`
- 引來你不想要的注意
- 比預期花更久時間

常見 13+ 額外收益：

- 佔到好位置，下一次擲骰 `+1 單次加值`
- 讓自己脫離當前危險
- 比預期更快完成，還能多做一件事

## 追獵（Hunt）

當你要追蹤怪物或人物、判斷是否有埋伏、或設下埋伏時，擲追獵。

- `6 以下`：你反被先手，或根本找不到目標。
- `7-9`：雙方互相發現，或你找到目標時已經太晚。
- `10-12`：你搶得先手或佔據先機。
- `13+`：你成功伏擊，且對目標第一次擲骰 `+1 單次加值`。

多人在同一情境各自擲追獵時，結果可不同：有人暴露、有人埋伏成功，也有人因衝刺而落後。
