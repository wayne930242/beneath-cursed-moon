# 動作

玩家透過「動作」與系統互動。任何人都可使用基本動作，而扮演書可能再提供額外專屬動作。

動作大致分三類：

- **戰鬥動作**：保護他人、壓制敵人（不一定只能在戰鬥中用）
- **調查動作**：花時間蒐集資訊
- **其他動作**：可跨情境使用，或單獨成立

每份扮演書都會給各動作對應加值。使用動作時擲 `2d6` 加上該加值。若你有**單次加值（Forward）**，套用在指定類型的下一次擲骰；若有**持續加值（Ongoing）**，則在條件成立期間持續套用。

## 戰鬥

### 浴血奮戰（Battle）

當你與某生物交戰，不論是想造成傷害，或阻止傷害落在自己或他人身上時，擲浴血奮戰。

- `6 以下`：你被輕易壓制。
- `7-9`：選 1。
- `10-12`：選 2。
- `13+`：選 3。

可選效果：

- 對對手造成傷害（依武器；額外再花 1 個選項可 `+1 傷害`）
- 讓對手失衡，使每位同伴對它獲得 `+1 單次加值`
- 從對手手中奪下／守住某個物品或位置
- 把對手注意力拉到你身上（離開其他人）
- 掙脫對手控制，或替他人掙脫
- 把對手逼到你要的位置，或阻止其脫離
- 同時攻擊多名敵人（此行動獲得 `group` 標籤）
- 到你下次行動前，對手對你造成的傷害 `-1`

### 理解戰況（Knowledge）

當你想在戰鬥中用建議協助同伴時，擲理解戰況。

- `6 以下`：你想不出有用資訊。
- `7-9`：獲得 `掌握 2`。
- `10-12`：獲得 `掌握 4`。
- `13+`：獲得 `掌握 6`。

你可隨時花 1 掌握來：

- 指出目標所有特殊能力與弱點
- 幫同伴找到破綻，使每位同伴對其 `+1 單次加值`
- 找到防禦缺口，使下一位攻擊該目標的同伴 `+1 傷害`
- 協助同伴避傷，使其承受傷害 `-1`
- 提供精神支撐或指引，使每位同伴下次抵抗黑魔法 `+1 單次加值`
